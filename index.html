Frequently Asked Questions

What is the difference between var, let, and const in JavaScript?

Answer: var, let, and const are used to declare variables in JavaScript. var is the older way and follows function scope. let and const were introduced in ES6 and follow block scope. The value of let can be changed, but const cannot be reassigned after declaration.


বাংলা: var, let এবং const JavaScript এ variable তৈরি করার জন্য ব্যবহার করা হয়। var পুরনো পদ্ধতি এবং এটি function scope অনুসরণ করে। let এবং const ES6 এ এসেছে এবং block scope অনুসরণ করে। let এর value পরিবর্তন করা যায়, কিন্তু const এর value পরিবর্তন করা যায় না।

What is the difference between map(), forEach(), and filter()?

Answer: map() creates a new array by changing every element. forEach() executes a function for each element but does not return a new array. filter() creates a new array with elements that match a specific condition.


বাংলা: map() প্রতিটি element এর উপর কাজ করে নতুন array তৈরি করে। forEach() প্রতিটি element এর উপর function চালায় কিন্তু নতুন array return করে না। filter() condition অনুযায়ী element বাছাই করে নতুন array তৈরি করে।

What are arrow functions and how are they different from regular functions?

Answer: Arrow functions are a shorter way to write functions in JavaScript. Unlike regular functions, arrow functions do not have their own this value. They get this from their surrounding scope.


বাংলা: Arrow function হলো JavaScript এ function লেখার সহজ পদ্ধতি। Regular function এর নিজস্ব this থাকে, কিন্তু arrow function এর নিজস্ব this থাকে না। এটি বাইরের scope থেকে this গ্রহণ করে।

How do JavaScript Promises work?

Answer: A Promise represents the success or failure of an asynchronous operation. It has three states: pending, fulfilled, and rejected. We use then() for success and catch() for errors.


বাংলা: Promise হলো asynchronous operation এর সফলতা বা ব্যর্থতার ফলাফল প্রকাশ করার একটি object। এর তিনটি state আছে: pending, fulfilled, এবং rejected। সফল ফলাফলের জন্য then() এবং error এর জন্য catch() ব্যবহার করা হয়।

How do closures work in JavaScript?

Answer: A closure allows a function to remember variables from its outer function even after the outer function has finished executing.


বাংলা: Closure এমন একটি feature যেখানে একটি function তার বাইরের function এর variable মনে রাখতে পারে, এমনকি বাইরের function execution শেষ হয়ে গেলেও।
